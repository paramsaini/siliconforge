=== CHECKPOINT: STA Industrial Upgrade ===
Date: 2026-03-11

TASK: Add 4 industrial STA features matching PrimeTime/Tempus capabilities.

IMPLEMENTED:

1. CPPR (Common Path Pessimism Removal)
   - Removes artificial pessimism from shared launch/capture clock paths
   - Algorithm: common_delay = min(launch_ins, capture_ins), credit = common × |late - early|
   - Setup: adds credit to slack; Hold: subtracts credit
   - References: DAC'21 Huang et al., Synopsys PrimeTime CPPR
   - API: enable_cppr(bool)

2. POCV (Parametric On-Chip Variation)
   - Per-cell sigma-based statistical derating (replaces flat/depth-based)
   - RSS path sigma: √(Σ σ_i²) for all cells on path
   - Final derate = mean ± N_sigma × path_sigma (default 3-sigma)
   - Per-cell sigma table with defaults (5%) and overrides
   - References: Synopsys POCV Application Note, LVF format
   - API: enable_pocv(n_sigma, default_sigma_pct), set_pocv_table()

3. PBA (Path-Based Analysis)
   - Re-evaluates critical paths with actual slew propagation
   - Path-specific delays instead of graph-wide worst-case
   - Integrates with POCV sigma and crosstalk delta
   - Reports both GBA and PBA results
   - API: enable_pba(bool)

4. Crosstalk Delta-Delay
   - Coupling capacitance from adjacent wire spacing/parallel run length
   - Delta = Cc × Miller_factor / slew
   - Configurable coupling cap, Miller factor, distance thresholds
   - References: Devgan, ICCAD 1997
   - API: enable_crosstalk(config) or enable_crosstalk(coupling_cap, miller)

FILES MODIFIED:
- src/timing/sta.hpp: Added OcvMode::POCV, PocvTable, CpprConfig, CrosstalkConfig structs;
  Extended TimingPath (cppr_credit, pba_slack, pba_delay, path_sigma, crosstalk_delta, pba_valid);
  Extended StaResult (cppr_total_credit, pba_wns/tns, max_crosstalk_delta, feature flags);
  New public methods: enable_cppr, enable_pocv, enable_pba, enable_crosstalk, set_pocv_table
