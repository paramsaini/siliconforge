================================================================================
                        KEY ALGORITHMS & TECHNIQUES
================================================================================

FRONTEND PARSING:
  • Verilog: 70+ token types, 12-level operator precedence, module elaboration
  • SDC: TCL-like constraint parsing with clock/exception handling
  • SVA: Linear Temporal Logic (LTL) with |-> and |=> operators
  • UPF: IEEE 1801 power domain hierarchy, isolation, level-shifting

CORE DATA STRUCTURES:
  • Logic4: IEEE 1364/1800 4-state (0/1/X/Z) with truth tables
  • Netlist: Gate-level with topological sorting, simulation support
  • AIG: AIGER encoding, structural hashing, DAG-aware construction
  • Liberty: NLDM 2D interpolation tables for timing/power
  • LEF: 5.8 standard layer/via/macro/pin definitions
  • 3D-IC: TSV parasitic physics (R/L/C), micro-bump models, yield analysis

OPTIMIZATION ALGORITHMS:
  • AIG Opt: DAG rewriting, tree balancing, SAT-based redundancy (Mishchenko)
  • Tech Map: Cut enumeration + area/delay covering (FlowMap, Cong & Ding)
  • Behavioral: 30 AST node types, multi-bit arithmetic, memory synthesis
  • Retiming: Leiserson-Saxe with hold-time, ICG preservation (Pan 1999)
  • Clock Gating: ICG insertion with enable factoring (Benini 1998)
  • Multi-Vt: Depth-based Vt assignment for leakage reduction
  • Resource Sharing: Mutual exclusivity analysis for area optimization
  • SAT: CDCL with 2-watched literals, First-UIP, VSIDS heuristic
