                        ALGORITHM & PHYSICS DETAILS
================================================================================

POWER ANALYSIS - ACTIVITY PROPAGATION:

Initial: All PIs and DFF outputs set to default_activity (default 0.1)

Per Gate Type:
  BUF/NOT:    α_out = avg(α_in)
  AND/NAND:   α_out = ∏(α_in)           [multiplicative]
  OR/NOR:     α_out = 1 - ∏(1-α_in)     [additive via complements]
  XOR/XNOR:   α_out = 2×α₁×(1-α₂)       [if 2 inputs]
  MUX:        α_out = avg(α_in)
  
Glitch Activity:
  Δα = base_activity × 0.08 × max(0, level_diff-1)   [capped at 50%]
  level_diff = max_input_level - min_input_level
  Detects reconvergent paths with timing skew

Power per Cell:
  P_leakage = cell_leakage(type) × 1e-6 × (Vdd/Vdd_ref)²
  P_switching = α × C_load × Vdd² × f
  P_internal = 0.15 × P_switching
  P_glitch = glitch_α × C_load × Vdd² × f
  P_total = P_leakage + P_switching + P_internal + P_glitch

IR DROP ANALYSIS - GAUSS-SEIDEL SOLVER:

Grid Discretization:
  • 16×16 mesh (configurable)
  • Boundary: VDD pads at all 4 edges, held at constant Vdd
  • Interior nodes: Floating voltage determined by current balance

Resistor Network:
  • Horizontal resistance: R_x = Rsheet × (dx/dy)
  • Vertical resistance:   R_y = Rsheet × (dy/dx)
  • Rsheet = 10 mΩ/sq (typical M8 power mesh)
  • dx, dy = grid cell width/height

Current Distribution:
  • I_sink[x][y] = sum of cell currents in each grid cell
  • If physical placement: Distribute per placed cell location
  • If no placement: Uniform distribution across grid
  • Total current = Power_mW / (Vdd_V × 1000) [in mA]

Solver Algorithm (Gauss-Seidel):
  for iter = 1 to 200:
    for each interior node (x,y):
      if (not on boundary):
        sum_g = 1/R_left + 1/R_right + 1/R_up + 1/R_down
        sum_gv = (V[x-1][y]/R_x) + (V[x+1][y]/R_x) + (V[x][y-1]/R_y) + (V[x][y+1]/R_y)
        V_new[x][y] = (sum_gv - I_sink[x][y]) / sum_g
        V_new[x][y] = min(V_new[x][y], Vdd)  // clamp to max
      delta_max = max(|V_new - V_old|)
    if delta_max < 1e-8: break

Convergence:
  • Max iterations: 200
  • Tolerance: 1e-8 V
  • Typically converges in 50-100 iterations

IR Drop Extraction:
  • Drop = Vdd - V[x][y]
  • Worst drop: max(Drop) across all nodes
  • Avg drop: mean(Drop) across interior nodes
  • Hotspots: Nodes where Drop > 5% × Vdd

================================================================================

PDN STRIPE TOPOLOGY:

Auto-Configuration:
  n_h_stripes = max(2, ceil(die_height / 20))
  n_v_stripes = max(2, ceil(die_width / 20))
  
  Horizontal stripes: y_offset = (i+0.5) × die_height / n_h_stripes
                      Layer: 3, Resistance: 0.05 Ω/um, Width: 1um
  
  Vertical stripes:   x_offset = (i+0.5) × die_width / n_v_stripes
                      Layer: 4, Resistance: 0.05 Ω/um, Width: 1um

Nearest Stripe Lookup:
  for each stripe s:
    if s is horizontal: dist = |y - s.offset|
    else:               dist = |x - s.offset|
    if dist < min_dist:
      min_dist = dist
      min_res = dist × s.resistance_per_um + pad_resistance

Pad Resistance: 0.01 Ω (typical BGA pad)

EM Violation Checking:
  current_density = I_local / stripe_width
  if current_density > 1.0 mA/μm: increment violation counter

================================================================================
                        FLOW INTEGRATION SEQUENCE
================================================================================

Typical Design Flow with IR Drop/Power Analysis:

1. read_verilog(file)
   └─ Creates nl_ (Netlist)
   
2. run_synthesis()
   └─ Maps to gate library
   
3. run_power()
   └─ PowerAnalyzer pa(nl_, &lib_);
   └─ result = pa.analyze(1.0 GHz, 1000mV default, 0.1 default_activity)
   └─ Stores: dynamic_mw, leakage_mw, switching_mw, total_mw
   
4. run_floorplan()
   └─ Creates pd_ (PhysicalDesign) with die_area
   
5. run_place()
   └─ Fills pd_.cells with placement coordinates
   └─ Sets c.placed = true for each cell
   
6. run_reliability()
   └─ ReliabilityAnalyzer ra(nl_, &pd_);
   └─ result = ra.analyze();           // NBTI/HCI/TDDB aging
   └─ ir_result = ra.analyze_ir_drop(power_result_.total_mw);
   │  ├─ Builds 16×16 grid
   │  ├─ Distributes current based on placed cells
   │  ├─ Runs Gauss-Seidel solver (200 iters max)
   │  └─ Extracts max/avg drops, hotspot count
   └─ Output: Aging + IR-drop results with warnings

Shell Commands:
  sf_shell> power           → Calls run_power()
  sf_shell> reliability     → Calls run_reliability()
  sf_shell> ir_drop         → Alias for reliability (IR-drop mesh part)
