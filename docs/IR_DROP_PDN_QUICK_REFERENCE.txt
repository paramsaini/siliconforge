KEY CLASSES & THEIR ROLES:
─────────────────────────────────────────────────────────────────────────────

1. PowerAnalyzer (power.hpp/cpp)
   ├─ analyze(freq_mhz, vdd, default_activity)
   ├─ Components: Dynamic + Static + Glitch + Clock power
   ├─ Activity Propagation: Gate-level with topo sort
   ├─ Glitch Estimation: Level-difference heuristic (capped 50%)
   ├─ File Support: VCD parsing (IEEE 1364-2001), SAIF parsing
   └─ Output: 10 top consumers, total mW breakdown, message

2. IrDropAnalyzer (ir_drop.hpp/cpp)
   ├─ analyze(grid_res=10)
   ├─ Model: Distance to 4 corner pads (simplified)
   ├─ Grid: Configurable resolution (default 10×10)
   ├─ Current Distribution: Proportional to cell area
   ├─ Hotspot Detection: >5% VDD drop threshold
   └─ Output: Worst/avg drops, hotspots, 2D drop map

3. PdnAnalyzer (pdn.hpp/cpp)
   ├─ auto_config(vdd, current_ma) | set_config(PdnConfig)
   ├─ analyze(grid_res=10)
   ├─ Model: Horizontal + Vertical stripe mesh
   ├─ Stripe Generation: ~Die_height/20 H-stripes, ~Die_width/20 V-stripes
   ├─ Resistance: Distance_to_stripe × Rsheet + Pad_R (typ 0.01Ω)
   ├─ EM Checking: Current density vs 1.0 mA/μm limit
   └─ Output: Worst/avg drops, EM violations, node voltage map

4. ReliabilityAnalyzer (reliability.hpp/cpp)
   ├─ analyze()  [NBTI/HCI/TDDB aging models]
   └─ analyze_ir_drop(power_mw)  [FULL RESISTIVE MESH SOLVER]
      ├─ Grid: 16×16 (fixed)
      ├─ Solver: Gauss-Seidel iterative (200 max iters, 1e-8 tolerance)
      ├─ Sheet Resistance: 10 mΩ/sq (top-level power mesh M8+)
      ├─ Boundary: VDD pads at all edges
      ├─ Current Sinks: Per-placed-cell distribution
      └─ Output: Max/avg drops, drop%, hotspot count & locations


ALGORITHM COMPLEXITY:
─────────────────────────────────────────────────────────────────────────────
PowerAnalyzer::analyze()         O(num_gates)          Activity propagation
PowerAnalyzer::glitch_activity() O(gate_fanin)         Topo level check
IrDropAnalyzer::analyze()        O(grid_res²)          Grid computation
PdnAnalyzer::analyze()           O(grid_res² × stripes) Nearest stripe lookup
ReliabilityAnalyzer::analyze_ir_drop() O(16² × 200)    Gauss-Seidel 200 iters

POWER COMPONENTS BREAKDOWN:
─────────────────────────────────────────────────────────────────────────────
total_power_mw = dynamic_power_mw + static_power_mw + clock_power_mw

dynamic_power_mw = switching_power_mw + internal_power_mw + glitch_power_mw

Per-Cell:
  - Leakage (static): Gate type lookup, V² scaling, cell_leakage() × vdd²
  - Switching: P = α × C × V² × f (activity from propagation)
  - Internal: 15% of dynamic (P_int = 0.15 × P_dyn)
  - Glitch: Reconvergent gate activity estimation

Clock:
  - DFF power: DFF count × full-toggle dynamic
  - CTS buffer: ~1 buffer per 8 DFFs, full toggle


HOTSPOT DETECTION THRESHOLDS:
─────────────────────────────────────────────────────────────────────────────
IrDropAnalyzer:           > 5% of VDD
PdnAnalyzer:              > 5% of VDD
ReliabilityAnalyzer:      > 5% of VDD (0.05 × vdd)
Flow Engine Warnings:     > 10% (reinforcement needed)


FILE FORMATS SUPPORTED:
─────────────────────────────────────────────────────────────────────────────
VCD (Value Change Dump):  IEEE 1364-2001, toggle count extraction
SAIF (Switching Activity): TC (toggle count), T0 (zero transitions), T1 (one)
Parsed by PowerAnalyzer for activity factors

DATA TYPES:
─────────────────────────────────────────────────────────────────────────────
PowerResult:
  ├─ Power (mW): total, dynamic, static, switching, internal, clock, glitch
  ├─ Frequency: clock_freq_mhz
  ├─ Voltage: supply_voltage
  ├─ Count: num_cells
  ├─ Timing: time_ms
  ├─ Top consumers: CellPower[] (name, dynamic, leakage, total)
  └─ Message: human-readable summary

IrDropResult:
  ├─ Drops (mV): worst, avg
  ├─ Hotspots: num_hotspots (count), hotspots[] (region, drop_mv, current_ma)
  ├─ Grid: drop_map[grid_y][grid_x], grid_x, grid_y
  ├─ Threshold: threshold_pct
  ├─ Voltage: vdd
  └─ Message: summary

PdnResult:
  ├─ Drops: worst_drop_mv, worst_drop_pct, avg_drop_mv
  ├─ EM: em_violations, worst_current_density
  ├─ Nodes: PdnNode[] (x, y, voltage, current_draw)
  └─ Timing: time_ms, message

AgingResult:
  ├─ Aging shifts: nbti_vth_shift_mv, hci_vth_shift_mv
  ├─ Reliability: tddb_failure_rate (FIT), em_failure_rate
  ├─ Timing: timing_degradation_pct
  ├─ Risk: cells_at_risk, details[] (CellAging[])
  ├─ Embedded IR-drop: IrDropResult with max_drop_mv, avg_drop_mv, drop_pct, hotspots
  └─ Timing: time_ms, message
