6 Industrial Optimization Passes (Phase 21):
1. Vt Swap (vt_swap_for_timing):
   HVT→SVT→LVT→ULVT based on path slack severity
2. High-Fanout Cloning (clone_high_fanout):
   clone drivers on critical paths
3. Buffer Insertion (insert_buffers_for_slew):
   fix slew/load violations

   CTS adjustment (opt-in, off by default)

  hold_buffers, useful_skew_adjustments, leakage_before/after, area_change_pct

STAGE 13 — Static Timing Analysis (timing/, 6,818 LOC)
================================================================================

STA Engine (StaEngine)
- Forward propagation: arrival times (AT) through topological order
- Backward propagation: required arrival times (RAT) from outputs
- TimingArc: from_pin, to_pin, delay, setup/hold
- PinTiming: arrival_early/late, required_early/late, slack
- TimingPath: startpoint, endpoint, path_delay, slack, depth, gates

OCV/AOCV (Phase 6):
- OcvMode: NONE, OCV, AOCV, POCV
- CornerDerate: cell_late, cell_early, wire_late, wire_early,
  clock_late, clock_early
- AocvTable: late/early_variation (default 10%),
  depth-dependent derating via sqrt(N) rule
- Gate depths via BFS from PIs/DFF outputs

Industrial STA (Phase 20):

  CPPR (Common Path Pessimism Removal):
  - CpprConfig: enabled flag
  - credit = common_delay x |late_derate - early_derate|
  - Setup: adds credit to slack; Hold: subtracts credit
  - Reference: DAC'21 Huang et al.

  POCV (Parametric On-Chip Variation):
  - PocvTable: default_sigma_pct (5%), n_sigma (3.0),
    per-cell sigma overrides
  - RSS path sigma: sqrt(sum(sigma_i^2))
  - Final derate = mean +/- N_sigma x path_sigma

  PBA (Path-Based Analysis):
  - Re-evaluates critical paths with actual slew propagation
  - Path-specific delays vs graph-wide worst-case
  - Reports GBA and PBA results

  Crosstalk Delta-Delay:
  - CrosstalkConfig: coupling_cap, miller_factor, distance_threshold
  - Delta = Cc x Miller_factor / slew
  - Reference: Devgan, ICCAD 1997

StaResult: wns, tns, paths, cppr_total_credit, pba_wns/tns,
  max_crosstalk_delta, feature flags
